What is this website

This website is a demonstration website made by the developer to both learn the intricacies of full stack website development. It was developed following the waterfall SDLC methodology and using the HTML, CSS, javascript, and react for the frontend, and node.js with express for the backend. Furthermore, the website used a mySQL database and is hosted using AWS services.

This websites design was made to be a marketplace in which individuals can easily list and purchase items in their possesions in a similar style to Gumtree and Ebay but with a focus on gaming merchandise such as board games, card games, video game equipment, and more. More about the design of the website including documentation of all stages of the SDLC can be found below.
